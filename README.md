# Calculator App 🧮

A simple and clean calculator built with HTML, CSS, and JavaScript, featuring a dark/light mode toggle.

## Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Dark / light mode toggle
- Clear/reset (C) button
- Minimal and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repository
```
git clone https://github.com/Alexia-Soare/calculator.git
```
2. Open `calculator.html` in your browser
3. Start calculating!
4. Toggle the 🌙 icon to switch between dark and light mode
